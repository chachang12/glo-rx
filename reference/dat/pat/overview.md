# DAT Perceptual Ability Test (PAT) — Section Overview

Stub for Phase 0 corpus-loader verification. Replace with real reference
material as authoring proceeds.

## Topic scope (90 questions, 60-minute section, 6 subskills)

- **Apertures (Keyholes)** — given a 3D object, identify which 2D opening it
  can pass through in ANY orientation without rotation during passage.
- **View Recognition (Top-Front-End)** — given two orthographic views of an
  object, pick the third.
- **Angle Discrimination** — rank four angles from smallest to largest.
- **Paper Folding** — predict the hole pattern after a folded paper is hole-
  punched and unfolded.
- **Cube Counting** — count cubes painted on N sides in a stacked figure.
- **3D Form Development (Pattern Folding)** — given a 2D net, pick the 3D
  shape it folds into.

## Question pattern notes

PAT is non-verbal — the items are diagrams. Reference material here helps
the AI write metadata (difficulty, subskill tags) and rationales; the
visual stems themselves are produced by separate authoring tooling. Until
that tooling exists, PAT generation is out of scope for AI question
synthesis.
