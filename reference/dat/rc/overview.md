# DAT Reading Comprehension — Section Overview

Stub for Phase 0 corpus-loader verification. Replace with real reference
material as authoring proceeds.

## Topic scope (50 questions across 3 passages, 60-minute section)

Passages are dense scientific or technical prose drawn from biology,
chemistry, anatomy, or applied health-science domains. The test does not
require outside knowledge — every answer is derivable from the passage.

Question types per passage:

- **Direct retrieval** — a fact stated explicitly in the text.
- **Tone & inference** — author intent, scoped to claims made in the
  passage.
- **Application** — apply a stated principle to a new scenario.
- **Search-and-locate** — keyword-anchored lookups across paragraphs.

## Question pattern notes

The dominant failure mode is time management, not comprehension. Search
strategies (skimming for keywords, mapping paragraph topics) outperform
linear reading. Generation should produce items whose answers are anchored
to specific spans of the source passage.
