# Reference: <EXAM NAME>

Paste the exam's subject material here. This is the same content you would
upload via Admin → Exam → AI Reference File.

Aim for a tight reference that includes:

- The topic outline as it appears in the admin Exam → Topics list. Use the
  exact topic strings — questions will be tagged with them and the admin
  filter matches verbatim.
- Style guidance specific to the exam (tone, scope, what counts as fair game,
  what's out of scope).
- Anchor facts you want questions tied to: specific drug names, exact lab
  values, named formulas, definitions, dates, eponyms.
- A short glossary if the exam has consistent abbreviations the model should
  use correctly.

Avoid:

- Dumping textbook chapters wholesale. The whole file goes into context every
  run — keep it focused.
- Generic "study tips" or test-taking advice. Those don't shape question
  content.
- Anything the model already knows from training. The reference is for
  *constraints* and *specifics*, not for teaching the subject.
